# SonetPay Tracker

A premium, responsive Kia Sonet payment and ownership tracker built as a static web app.

## Run Locally

```powershell
npm.cmd run start
```

Then open:

```text
http://127.0.0.1:4173/
```

## Deploy To GitHub Pages

This app is static, so GitHub Pages can serve it directly from the repository root.

1. Push this folder to a GitHub repository.
2. Open the repository on GitHub.
3. Go to `Settings` -> `Pages`.
4. Under `Build and deployment`, choose `Deploy from a branch`.
5. Select the `main` branch and `/ (root)` folder.
6. Save. GitHub will publish the site at the Pages URL shown there.
